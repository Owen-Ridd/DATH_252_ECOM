## How to Download from Sketchfab

1. Click on a model
2. Click "Download 3D Model" button
3. Select "glTF" format (not Binary glTF)
4. Extract the ZIP file
5. Find the `.glb` file
6. Place it in this directory

## File Naming Convention

- Use lowercase with underscores
- Format: `{category}_{style}.glb`
- Examples: `sofa_modern.glb`, `chair_leather.glb`, `table_wood.glb`
